### Linux Setups
1. clone this repo ```sh git clone https://github.com/NoerNova/machine.env.git```
2. Navigate to folder ```sh cd machine.env```
3. ```sh chmod +x linux-full-setups.sh```
4. ```sh linux-full-setups.sh``` or ```sh ./linux-full-setups.sh```
